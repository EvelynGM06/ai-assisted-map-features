## Project Overview

This project develops a computer vision pipeline for the detection of community relevant map features from high-resolution satellite imagery. The workflow combines geospatial data processing, dataset preparation, object detection and evaluation techniques to support urban analysis and risk assessment applications.

---

## Objective

The objective of this project is to automate the identification of urban structures from satellite imagery using deep learning techniques and geospatial analysis tools.

---

## Methodology

1. Orthophoto acquisition and preprocessing.
2. Image tiling and dataset preparation.
3. Bounding box annotation.
4. YOLO model training.
5. Prediction and evaluation.
6. Geospatial analysis using QGIS.
